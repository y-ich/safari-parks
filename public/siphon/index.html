Siphon: CoffeeScript Programming Environment on iPad

Enjoy from here!

Features

  • Real-time compiling same as the CoffeeScript site
  • Software keyboard enhancement
    • NOTE: You can input single or double quatation by flicking comma(') or period(.) key on software keyboard respectively. So there is no enhancement for them.
  • "File" storage
  • Offline application
  • jQuery library

Action Items

  • Click sound by clicking button
  • Pretty printing
  • Editor enhancement
  • Prelude library in "Smooth CoffeeScript"
  • Canvas/SVG libray
